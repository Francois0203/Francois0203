# Not Found

A visually appealing 404 error page component with animated effects and navigation options.

## Features

- Animated 404 display with glitch effects
- Gradient background with geometric patterns
- Clear error messaging
- Navigation buttons (Go Home, Go Back)
- Responsive design
- Accessibility features
- Smooth transitions

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| None | - | - | This component doesn't accept props |

## Usage

```jsx
import NotFound from './pages/Not Found';

function App() {
  return (
    <div>
      <NotFound />
    </div>
  );
}
```
